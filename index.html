How much do people trust AI healthcare answers?

This prototype runs a controlled live study: participants review fixed healthcare prompts, generate an AI model answer, rate how trustworthy it feels, and compare that trust against a clinician note.

Start Study View Method

Study Focus

  • Trust How believable the AI feels
  • Safety Whether users would act on it
  • Confidence How certain the user feels

Goal

Measure trust, not just accuracy.

People may trust an answer even when it is incomplete or risky. This interface tests that gap by showing AI advice first, then a clinician guidance benchmark.

Method

Twelve healthcare scenarios.

Each participant reads the same twelve fixed prompts, requests a live AI model answer for each one, rates it on three scales, and explains their decision in a short reflection.

Output

Instant local summary.

The site calculates average trust levels across the scenarios so you can discuss patterns during your class presentation.
